### Example Usage

```hcl
provider "github" {
  version = "4.1.0"

  # base_url - (optional) is a type of string
  base_url = null
  # insecure - (optional) is a type of bool
  insecure = null
  # organization - (optional) is a type of string
  organization = null
  # owner - (optional) is a type of string
  owner = null
  # token - (optional) is a type of string
  token = null
}
```
